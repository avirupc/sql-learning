### 1. Show the population of Germany.
```sql
SELECT
    population
FROM
    world
WHERE
    name = 'Germany' 
```

Note: strings should be in 'quotes'

### 2. Show the name and the population for 'Sweden', 'Norway' and 'Denmark'.
Checking a list: The word `IN` allows us to check if an item is in a list. 
```sql
SELECT
    name,
    population
FROM
    world
WHERE
    name IN ('sweden', 'Norway', 'Denmark')
```
Note: SQL is not case sensitive. I have used 'sweden' instead of 'Sweden' and it still works!

### 3,  Show the country and the area for countries with an area between 200,000 and 250,000.

`BETWEEN` allows range checking (range specified is inclusive of boundary values). 

```sql
SELECT
    name,
    area
FROM
    world
WHERE
    area BETWEEN 200000
    AND 250000;
```
