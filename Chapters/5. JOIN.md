### 1. Show the matchid and player name for all goals scored by Germany.

<details>
<summary>🟩Solution</summary>

```SQL
SELECT matchid, player FROM goal
WHERE teamid = 'GER'
```
</details>

### 2. From the previous query you can see that Lars Bender's scored a goal in game 1012. Now we want to know which teams were playing in that match.

Notice in the that the column `matchid` in the goal table corresponds to the `id` column in the game table. We can look up information about game 1012 by finding that row in the game table.

<details>
<summary>🟩Solution</summary>

```SQL
SELECT id,stadium,team1,team2
FROM game
WHERE id = 1012
```
</details>

You can combine the two steps into a single query with a `JOIN`. 

```SQL
SELECT *
FROM game JOIN goal ON (id=matchid)
```
The ``FROM`` clause says to merge data from the goal table with that from the game table. The `ON` says how to figure out which rows in game go with which rows in goal - the `matchid` from goal must match `id` from game. (If we wanted to be more clear/specific we could say
`ON (game.id=goal.matchid)`)

### 3. Show the player, teamid, stadium and mdate for every German goal

<details>
<summary>🟩Solution</summary>

```SQL
SELECT
    player, teamid, stadium, mdate
FROM
    game
    JOIN goal ON (id = matchid)
WHERE
    teamid = 'GER'
```
</details>
 
### 4. Show the team1, team2 and player for every goal scored by a player called Mario

<details>
<summary>🟩Solution</summary>

```SQL
SELECT team1, team2, player
FROM game JOIN goal ON (id=matchid)
WHERE player LIKE 'Mario%'
```
</details>

### 5. Show player, teamid, coach, gtime for all goals scored in the first 10 minutes gtime<=10

<details>
<summary>🟩Solution</summary>

```SQL
SELECT player, teamid, coach, gtime
FROM goal JOIN eteam on teamid=id
WHERE gtime<=10
```
</details>

### 6. List the dates of the matches and the name of the team in which 'Fernando Santos' was the team1 coach

<details>
<summary>🟩Solution</summary>

```SQL
SELECT mdate, teamname
FROM eteam JOIN game on (team1=eteam.id)
WHERE coach = 'Fernando Santos'
```
</details>

<br>
⚠️**Note:**<br>
Notice that because `id` is a column name in both game and eteam you must specify `eteam.id` instead of just `id`.

### 7. List the player for every goal scored in a game where the stadium was 'National Stadium, Warsaw'

<details>
<summary>🟩Solution</summary>

```SQL
SELECT player
FROM game JOIN goal ON (id = matchid)
WHERE stadium = 'National Stadium, Warsaw'
```
</details>

### 8. Show the name of all players who scored a goal against Germany.

<details>
<summary>🟩Solution</summary>

```SQL
SELECT DISTINCT g.player
FROM game AS gm
JOIN goal AS g ON g.matchid = gm.id
WHERE (gm.team1 = 'GER' OR gm.team2 = 'GER') --  or: WHERE 'GER' IN (gm.team1, gm.team2)
  AND g.teamid <> 'GER'
```
</details>

### 9. Show teamname and the total number of goals scored.

<details>
<summary>🟩Solution</summary>

```SQL
SELECT teamname, COUNT(gtime)
FROM eteam JOIN goal ON id=teamid
GROUP BY teamname
```
</details>

### 10. Show the stadium and the number of goals scored in each stadium. 

<details>
<summary>🟩Solution</summary>

```SQL
SELECT stadium, COUNT(gtime)
FROM game JOIN goal ON id=matchid
GROUP BY stadium
```
</details>

### 11. For every match involving 'POL', show the matchid, date and the number of goals scored.

<details>
<summary>🟩Solution</summary>

```SQL
SELECT 
    matchid, mdate, COUNT(gtime)
FROM 
    game 
JOIN 
    goal 
    ON id = matchid 
WHERE 
    team1 = 'POL' 
    OR team2 = 'POL'
GROUP BY 
    matchid, mdate;
```
</details>

<br>
⚠️**Note:**<br>

In SQL, every column in the `SELECT` clause must either be:
1. an aggregate function (like `COUNT`, `SUM`, etc.), or
2. included in the `GROUP BY` clause

That's why, we had to include **both** `matchid` and `mdate` in `GROUP BY` in the above query. Otherwise, there would be an error.

### 12. For every match where 'GER' scored, show matchid, match date and the number of goals scored by 'GER'

<details>
<summary>🟩Solution</summary>

```SQL
SELECT matchid, mdate, COUNT(gtime)
FROM game 
JOIN goal ON id = matchid 
WHERE teamid = 'GER'
GROUP BY matchid, mdate
HAVING COUNT(gtime) > 0
```
</details>
