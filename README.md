# 📘 Learning SQL 

Hi all! 👋
I am trying to build a repository for **beginners who want to learn SQL from scratch**.

This repo will contain **chapter-wise tutorials** mostly in a **question → answer format**. The idea is to learn SQL by **looking at problems, writing queries, and understanding the solutions**.

---

## 📚 Learning Resources I Am Following

I am closely following these two excellent online resources:

1. https://sqlzoo.net/wiki/SQL_Tutorial (this is the primary one I am following)
2. https://www.sql-practice.com/

Both of these platforms are **fantastic for learning the fundamentals of SQL and practising queries**.

The goal of this repository is to **collect and organize problems, explanations, and solutions in one place** so that someone going through this repo can gradually build a **clear understanding of SQL syntax and common query patterns**.
